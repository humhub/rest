Add a new user to the space

path Parameters
id
required
integer

The id of space

userId
required
integer

The id of user

query Parameters
silent
boolean
Default: false

Send added notification to the user

canLeave
integer
Default: 1

0: user cannot cancel membership | 1: can cancel membership | 2: depending on space flag members_can_leave

showAtDashboard
boolean
Default: true

Show at dashboard

Responses

/api/v1/space/{id}/membership/{userId}
